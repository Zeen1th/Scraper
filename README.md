<div style="border: 2px solid #4CAF50; padding: 10px; margin: 10px 0;">

    <li><strong>Scrape and Save:</strong> Extract text content and save it to a file with a single click.</li>
    <li><strong>Error Handling:</strong> Alerts you if an error occurs during the scraping process.</li>

    <li><strong>Enter URL:</strong></li>
    <pre><code>A GUI will appear. Enter the URL of the webpage you wish to scrape.</code></pre>
    <li><strong>Scrape and Save:</strong></li>
